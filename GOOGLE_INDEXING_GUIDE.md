## Google'da Neden Çıkmıyor?

### 1. Site Yeni
- Google'ın yeni siteleri indekslemesi 1-4 hafta sürebilir
- İlk indeksleme sonrası düzenli tarama başlar

### 2. Yapılması Gerekenler

#### A. Google Search Console Kurulumu (ÖNEMLİ!)
1. https://search.google.com/search-console adresine git
2. "Mülk Ekle" butonuna tıkla
3. Domain veya URL prefix seçeneğinden birini seç
4. Doğrulama yöntemlerinden birini kullan:
   - HTML dosyası yükleme (public/google70c0c8b7abc1c3fb.html zaten var)
   - HTML meta etiketi
   - Google Analytics
   - Google Tag Manager

#### B. Sitemap Gönderimi
1. Search Console'da "Sitemaps" bölümüne git
2. Sitemap URL'ini ekle: `https://yksnethesapla.com/sitemap.xml`
3. "Gönder" butonuna tıkla
4. Google sitemap'i işlemeye başlayacak

#### C. URL İnceleme Aracı
1. Search Console'da "URL İnceleme" aracını kullan
2. Ana sayfayı test et: `https://yksnethesapla.com`
3. "Canlı Test İste" butonuna tıkla
4. "İndeksleme İste" butonuna tıkla
5. Önemli sayfalar için tekrarla:
   - /blog
   - /blog/yks-net-hesaplama-nasil-yapilir
   - /geri-sayim
   - /sss

### 3. SEO İyileştirmeleri

#### A. Backlink Oluşturma

- YKS ile ilgili sitelere yorum yap (link bırak)
- Reddit, Ekşi Sözlük gibi platformlarda paylaş

#### B. İçerik Güncellemeleri

- Yeni makaleler ekle (haftada 1-2)
- Kullanıcı yorumları ekle
- SSS bölümünü genişlet
